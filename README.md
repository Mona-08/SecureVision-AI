# 🛡️ SecureVision AI
**Sports Integrity & Media Verification Scanner**

### 🔗 Links
- **Live Demo:**  https://vision-ai-e3887.web.app
- **Backend Space:** https://mona-08-securevision-ai.hf.space

### 🚀 Features
- **Logo Detection:** Identifies official sports broadcasters.
- **Integrity Analysis:** Uses gemini-2.5-flash to verify sports news claims.
- **Multi-Media Support:** Scans text, websites, and YouTube links.

### 🛠️ Tech Stack
- **Frontend:** Firebase Hosting, Tailwind CSS
- **Backend:** Flask, Google Generative AI (Gemini)
- **Deployment:** Docker, Hugging Face Spaces
